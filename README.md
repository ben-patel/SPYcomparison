## Dependencies

To use the project, you'll need to install the following dependencies:
- c++11
- cmake >= 3.6.3
- curl
- python3
- pip

## Build

Build the code by running the following commands in order:
```
mkdir build
cd build
cmake ..
make
```

Setup the code by running:
```
export PATH=$PATH:${pwd}
chmod u+x CompareToSPY
```

## Run Code

Run the code with:
```
CompareToSPY <SYMBOL>
```

e.g.
```
CompareToSPY NVDA
```
